Simple Byte Sequence Patcher

    SBSP Version 0.0.0a

    No Decompressor/Compressor support


Simple Byte Sequence Patcher is a web GUI as template for a context (game rom or app bin file).

    SBSP source code :

        Modify main source code in repo/src

        Rebuild by running repo/build.bat

        Get result in repo/output

            core/default.css
            core/SBSP.js
            user/custom.css
            user/config.js
